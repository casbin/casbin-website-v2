## Wie kann ich Casbin als Service nutzen?

| Name                                                       | Beschreibung                                                                                                                                      |
| ---------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Kasbin-Server](https://github.com/casbin/casbin-server)   | Die offizielle `Casbin als Service` Lösung basierend auf [gRPC](https://grpc.io/), sowohl Management API als auch RBAC API werden bereitgestellt. |
| [middleware-acl](https://github.com/luk4z7/middleware-acl) | Wiederherstellende Zugriffssteuerung Middleware basierend auf Casbin.                                                                             |
| [Tasten](https://github.com/merajsahebdar/buttress-server) | Die Access Control als Service Lösung basierend auf Casbin.                                                                                       |
| [auth-Server](https://github.com/ZettaAI/auth-server)      | Auth Server zum Korrekturlesen von Diensten.                                                                                                      |
