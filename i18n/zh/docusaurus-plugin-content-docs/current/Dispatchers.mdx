**调度器（Dispatcher）** 将**增量**策略更新传播到多个执行器实例（例如集群）。它们使用一致性协议（例如 Raft），所有执行器最终看到相同的策略。Casbin 的正常 API（AddPolicy、RemovePolicy 等）保持不变；调度器将更新发送到其他节点。

两部分：(1) Casbin 内部由调度器实现的钩子，以及 (2) 调度器特定的设置（例如集群成员资格、配置）。

:::note
调度器仅在初始化后同步**更改**。它们**不**修复现有的分歧。在启用调度器之前，请确保所有执行器从相同的策略开始（例如，来自同一个数据库或快照）。
:::
