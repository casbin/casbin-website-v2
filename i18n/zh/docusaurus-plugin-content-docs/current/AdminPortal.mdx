**[Casdoor](https://casdoor.org)** 是一个基于 Web 的身份与访问平台，可以管理 Casbin 的模型（Model）和策略（Policy）：

![model editor](/img/docs/admin-portal-model-editor.png)

![policy editor](/img/docs/admin-portal-policy-editor.png)

下表列出了使用 Casbin 进行授权（Authorization）的**第三方管理门户和应用项目**。它们可作为你自己服务的参考或起点。
