Casbin 中间件（Middleware）将授权（Authorization）集成到 Web 框架和 API 网关中。它们通常将每个请求映射到 Casbin 的 `Enforce(sub, obj, act)` 调用，这样您就可以保护路由和资源，而无需编写样板代码。
