### GetImplicitUsersForResourceByDomain

返回给定资源和域的隐式用户（及其权限），即通过其角色在该域中拥有资源权限的用户。

```csv
p, admin, domain1, data1, read
p, admin, domain1, data1, write
p, admin, domain2, data2, read
p, admin, domain2, data2, write
g, alice, admin, domain1
g, bob, admin, domain2
```

示例：使用下面的策略，`GetImplicitUsersForResourceByDomain("data1", "domain1")` 返回可以访问 `domain1` 中 `data1` 的用户的权限（例如 alice 通过 admin 角色）。

```mdx-code-block
<Tabs groupId="langs">
<TabItem value="Go" label="Go" default>
```

```go
ImplicitUsers, err := e.GetImplicitUsersForResourceByDomain("data1", "domain1")
```

```mdx-code-block
</TabItem>
</Tabs>
```

:::note
仅返回用户级别结果；角色名称（`g` 规则中的第二个元素）不会被列出。
:::
