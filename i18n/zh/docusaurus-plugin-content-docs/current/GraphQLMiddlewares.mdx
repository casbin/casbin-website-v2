根据 [GraphQL 的授权指南](https://graphql.org/learn/authorization/)，授权应集中在一个地方——即 GraphQL 层和解析器之间。在解析器（或一个轻量包装器）中使用 Casbin 来对类型、字段或操作（Operation）实施访问控制。

```javascript
// Casbin 授权逻辑位于 postRepository 内部
var postRepository = require(‘postRepository’);

var postType = new GraphQLObjectType({
  name: ‘Post’,
  fields: {
    body: {
      type: GraphQLString,
      resolve: (post, args, context, { rootValue }) => {
        return postRepository.getBody(context.user, post);
      }
    }
  }
});
```
