本页报告了不同模型（Model）和规则规模下的**执行开销（Enforcement Overhead）**（每次 `Enforce()` 调用的时间和内存）。结果取决于硬件和语言，请仅作大致参考。

```mdx-code-block
<Tabs groupId="langs">
<TabItem value="Go" label="Go" default>
```

Go基准测试：[model_b_test.go](https://github.com/casbin/casbin/blob/master/model_b_test.go)。环境：

```text
Intel(R) Core(TM) i7-6700HQ CPU @ 2.60GHz, 2601 Mhz, 4 Core(s), 8 Logical Processor(s)
```

`go test -bench=. -benchmem` 的结果（op = 一次 `Enforce()` 调用，ms = 毫秒，KB = 千字节）：

| 测试用例                   | 规则规模                                    | 时间开销 (ms/op) | 内存开销 (KB) |
|---------------------------|--------------------------------------------|------------------|---------------|
| ACL                       | 2条规则（2个用户）                          | 0.015493         | 5.649         |
| RBAC                      | 5条规则（2个用户，1个角色）                 | 0.021738         | 7.522         |
| RBAC (small)              | 1100条规则（1000个用户，100个角色）         | 0.164309         | 80.620        |
| RBAC (medium)             | 11000条规则（10000个用户，1000个角色）      | 2.258262         | 765.152       |
| RBAC (large)              | 110000条规则（100000个用户，10000个角色）   | 23.916776        | 7,606         |
| RBAC with resource roles  | 6条规则（2个用户，2个角色）                 | 0.021146         | 7.906         |
| RBAC with domains/tenants | 6条规则（2个用户，1个角色，2个域）          | 0.032696         | 10.755        |
| ABAC                      | 0条规则（0个用户）                          | 0.007510         | 2.328         |
| RESTful                   | 5条规则（3个用户）                          | 0.045398         | 91.774        |
| Deny-override             | 6条规则（2个用户，1个角色）                 | 0.023281         | 8.370         |
| Priority                  | 9条规则（2个用户，2个角色）                 | 0.016389         | 5.313         |
