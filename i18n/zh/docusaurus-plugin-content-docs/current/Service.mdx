## 如何使用Casbin作为服务？

| 名称                                                           | 描述                                                                               |
| ------------------------------------------------------------ | -------------------------------------------------------------------------------- |
| [Casbin服务](https://github.com/casbin/casbin-server)          | 基于[gRPC](https://grpc.io/)的官方`Casbin as a Service`，提供了Management API 和 RBAC API。 |
| [middleware-acl](https://github.com/luk4z7/middleware-acl)   | 基于 Casbin 的 RESTful 访问控制插件。                                                      |
| [Buttress](https://github.com/merajsahebdar/buttress-server) | 基于Casbin访问控制的服务解决方案。                                                             |
| [auth-server](https://github.com/ZettaAI/auth-server)        | 验证服务器校对服务。                                                                       |
