## 匹配器中的函数

您可以在匹配器中使用内置函数或注册自定义函数。内置的键匹配函数具有以下形式：

```c
bool function_name(string url, string pattern)
```

它们返回 `url` 是否匹配 `pattern`。

可用的内置函数：

| 函数        | url                                           | pattern                                                          | 示例                                                                                                                                                                                               |

键提取函数接受三个参数（`keyGet` 除外，它只接受两个）：

```c
bool function_name(string url, string pattern, string key_name)
```

当 URL 匹配模式时返回命名键的值，否则返回 `""`。

示例：

- `KeyGet2("/resource1/action", "/:res/action", "res")` → `"resource1"`
- `KeyGet3("/resource1_admin/action", "/{res}_admin/*", "res")` → `"resource1"`
- `KeyGet("/resource1/action", "/*")` → `"resource1/action"`（双参数形式）

## 添加自定义函数

1. 实现一个接受所需参数并返回 `bool` 的函数：

    ```go
    func KeyMatch(key1 string, key2 string) bool {
        i := strings.Index(key2, "*")
        if i == -1 {
            return key1 == key2
        }

        if len(key1) > i {
            return key1[:i] == key2[:i]
        }
        return key1 == key2[:i]
    }
    ```

2. 为 Casbin 包装该函数（签名 `func(...interface{}) (interface{}, error)`）：

    ```go
    func KeyMatchFunc(args ...interface{}) (interface{}, error) {
        name1 := args[0].(string)
        name2 := args[1].(string)

        return (bool)(KeyMatch(name1, name2)), nil
    }
    ```

3. 在执行器上注册：

    ```go
    e.AddFunction("my_func", KeyMatchFunc)
    ```

4. 在模型中使用：

    ```ini
    [matchers]
    m = r.sub == p.sub && my_func(r.obj, p.obj) && r.act == p.act
    ```
