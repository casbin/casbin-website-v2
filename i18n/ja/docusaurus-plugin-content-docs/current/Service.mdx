## CasbinをServiceで使用する方法

| 名前                                                         | 説明                                                                                  |
| ---------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| [カスビンサーバー](https://github.com/casbin/casbin-server)        | `カスビンは` gRPC [に基づくサービス](https://grpc.io/)ソリューションとして、管理 API と RBAC API の両方が提供されています。 |
| [middleware-acl](https://github.com/luk4z7/middleware-acl) | Casbinに基づくRESTfulアクセス制御ミドルウェア。                                                      |
| [バットレス](https://github.com/merajsahebdar/buttress-server)  | Casbinに基づくサービスソリューションとしてのアクセス制御。                                                    |
| [auth-server](https://github.com/ZettaAI/auth-server)      | 校正サービスの認証サーバ。                                                                       |
