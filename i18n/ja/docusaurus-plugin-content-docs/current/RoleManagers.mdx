ロールマネージャは、Casbin内のRBACロール階層(ユーザーロールマッピング)を管理するために使用されます。 ロールマネージャは、CasbinポリシールールまたはLDAP、Okta、Auth0、Azure ADなどの外部ソースからロールデータを取得できます。 ロールマネージャの実装をサポートします。 軽量を維持するために、メインライブラリにロールマネージャーのコードを入れません(デフォルトのロールマネージャを除く)。 Casbin ロールマネージャの完全なリストは以下のとおりです。 新しいロールマネージャーに対するサードパーティの貢献を歓迎します。お知らせください。このリストに記載します:)

```mdx-code-block
<Tabs groupId="langs">
<TabItem value="Go" label="Go" default>
```

| ロールマネージャー                                                                                                        | 作成者                                           | 説明                                                                                                                                    |
| ---------------------------------------------------------------------------------------------------------------- | --------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| [デフォルトのロールマネージャー（ビルトイン）](https://github.com/casbin/casbin/blob/master/rbac/default-role-manager/role_manager.go) | Casbin                                        | Casbin ポリシーに格納されたロール階層をサポート                                                                                                           |
| [セッションロールマネージャー](https://github.com/casbin/session-role-manager)                                                 | [EDOMO システム](https://github.com/edomosystems) | Casbin ポリシーに保存されているロール階層をサポートし、時間範囲ベースのセッションを使用します。                                                                                   |
| [Okta ロールマネージャー](https://github.com/casbin/okta-role-manager)                                                    | Casbin                                        | [Okta](https://www.okta.com/) に格納されたロール階層をサポート                                                                                        |
| [Auth0 ロールマネージャー](https://github.com/casbin/auth0-role-manager)                                                  | Casbin                                        | [Auth0](https://auth0.com/)の [Authorization Extension](https://auth0.com/docs/extensions/authorization-extension/v2) に格納されたロール階層をサポート |
