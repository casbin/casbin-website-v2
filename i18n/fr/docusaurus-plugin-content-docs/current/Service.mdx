## Comment utiliser Casbin comme service?

| Nom                                                                | Description                                                                                                                              |
| ------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------- |
| [Serveur Casbin](https://github.com/casbin/casbin-server)          | La solution officielle `Casbin as a Service` basée sur [gRPC](https://grpc.io/), à la fois l'API de gestion et l'API RBAC sont fournies. |
| [acl middleware](https://github.com/luk4z7/middleware-acl)         | RESTful access control middleware based on Casbin.                                                                                       |
| [Bouton](https://github.com/merajsahebdar/buttress-server)         | La solution de contrôle d'accès en tant que service basé sur Casbin.                                                                     |
| [serveur-authentification](https://github.com/ZettaAI/auth-server) | Serveur d'authentification pour les services de relecture.                                                                               |
