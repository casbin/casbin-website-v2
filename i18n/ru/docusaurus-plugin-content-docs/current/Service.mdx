## Как использовать Casbin в качестве сервиса?

| Наименование                                                    | Описание                                                                                                              |
| --------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------- |
| [Сервер Casbin](https://github.com/casbin/casbin-server)        | Представлено официальное решение `Casbin как сервис` на основе [gRPC](https://grpc.io/), и Management API и RBAC API. |
| [среднее ПО акл](https://github.com/luk4z7/middleware-acl)      | Система контроля доступа middleware на базе Casbin.                                                                   |
| [Баттерс](https://github.com/merajsahebdar/buttress-server)     | Управление доступом как сервисное решение, основанное на Casbin.                                                      |
| [сервер аутентификации](https://github.com/ZettaAI/auth-server) | Сервер аутентификации для корректоров.                                                                                |
