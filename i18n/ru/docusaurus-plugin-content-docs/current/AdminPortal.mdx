Мы предоставляем [веб-портал под названием Casdoor](https://casdoor.org) для управления моделями и управления политикой:

![редактор модели](https://hsluoyz.github.io/casbin/ui_model_editor.png)

![редактор политики](https://hsluoyz.github.io/casbin/ui_policy_editor.png)

Существует также сторонние проекты админ-портала, которые используют Casbin в качестве движка авторизации. Вы можете начать создавать свой собственный сервис Casbin, основываясь на этих проектах.
