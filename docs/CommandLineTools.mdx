**Casbin Command Line Tools (CLIs) provide a command line interface for Casbin, enabling you to use all Casbin APIs in the shell. This document covers Casbin CLI usage for multiple languages including Rust, Java, Go, and Node.js.

## Installation

### Rust (casbin-rust-cli)

#### Installation Steps

1. Install via cargo

    ```bash
cargo install casbin-cli
```

2. Verify installation

    ```bash
casbin-cli --help
```

### Node.js (casbin-nodejs-cli)

#### Installation Steps

1. Install via npm

    ```bash
npm install -g casbin-nodejs-cli
```

2. Verify installation

    ```bash
casbin-cli --help
```

### Java (casbin-java-cli)

#### Installation Steps

1. Download the latest JAR package

    ```bash
wget https://github.com/casbin/casbin-java-cli/releases/latest/download/casbin-java-cli.jar
```

2. Verify installation

    ```bash
java -jar casbin-java-cli.jar --help
```

### Go (casbin-go-cli)

1. Install Go version CLI

    ```bash
go install github.com/casbin/casbin-go-cli/v2@latest
```

2. Verify installation

    ```bash
casbin-cli --help
```

### Go (casbin-go-cli)

1. Clone repository

    ```bash
git clone https://github.com/casbin/casbin-go-cli.git
```

2. Enter directory and build

    ```bash
cd casbin-go-cli
go build -o casbin-cli
# Verify installation
casbin-cli --help
```

### Rust (casbin-rust-cli)

#### From crates.io

```bash
cargo install casbin-rust-cli
```

#### From source

1. Clone repository

    ```bash
git clone https://github.com/casbin/casbin-rust-cli.git
```

2. Enter directory and build

    ```bash
cd casbin-rust-cli
cargo build --release
# Verify installation
./target/release/casbin-rust-cli --help
```

### Java (casbin-java-cli)

1. Clone repository

    ```bash
git clone https://github.com/jcasbin/casbin-java-cli.git
```

2. Build project (JAR package will be generated in target directory)

    ```bash
cd casbin-java-cli
mvn clean install
# Verify installation
java -jar target/casbin-java-cli.jar --help
```

## Usage Instructions

Casbin CLI tools provide core functions such as policy checking and policy management, supporting specification of model files, policy files, and operation instructions through command line parameters. CLI tools implemented in different languages have consistent basic usage. The following is a general operation guide.

## Options

| Option                 | Description                                                         | Required | Notes (Parameter Format Example) |
|------------------------|----------------------------------------------------------------------|----------|----------------------------------|
| `-m, --model`          | Model file path or inline model text                                | Yes      | Use `""` to wrap parameter values, multi-line content separated by `\|` (e.g., `"line1\|line2"`) |
| `-p, --policy`         | Policy file path or inline policy text                              | Yes      | Use `""` to wrap parameter values, multi-line content separated by `\|` (e.g., `"line1\|line2"`) |
| `-e, --enforce`        | Check permissions                                                    | No       | Use `""` to wrap parameter values |
| `-ex, --enforceEx`     | Check permissions and return matching policies                      | No       | Use `""` to wrap parameter values |
| `-AF, --addFunction`   | Add custom functions (only supported by casbin-java-cli)            | No       | Use `""` to wrap parameter values, multi-line content separated by `\|` (e.g., `"line1\|line2"`) |
| `-ap, --addPolicy`     | Add policy rules to policy file (only supported by casbin-java-cli) | No       | Use `""` to wrap parameter values, multi-line content separated by `\|` (e.g., `"line1\|line2"`) |
| `-rp, --removePolicy`  | Remove policy rules from policy file (only supported by casbin-java-cli) | No    | Use `""` to wrap parameter values, multi-line content separated by `\|` (e.g., `"line1\|line2"`) |
