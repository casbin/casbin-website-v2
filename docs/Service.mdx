## How to Use Casbin as a Service?

| Name                                                         | Description                                                                                                                                 |
|--------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [Casbin Server](https://github.com/casbin/casbin-server)     | The official "Casbin as a Service" solution built on [gRPC](https://grpc.io/). Provides both Management API and RBAC API.                 |
| [middleware-acl](https://github.com/luk4z7/middleware-acl)   | RESTful access control middleware powered by Casbin.                                                                                        |
| [auth-server](https://github.com/ZettaAI/auth-server)        | Authentication server for proofreading services.                                                                                            |
